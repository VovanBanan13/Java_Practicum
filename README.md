# Java_Practicum
Проект для практикума Java BackEnd

## Работа со складом новогодних игрушек
`PostgreSQL`  `Liquibase` `Hibenate` `Swagger API` `Maven` `Spring MVC` `Spring Boot` `JSON`
### Функции приложения
- Управление складом:
    - Добавление 
    - Поиск
    - Обновление
    - Удаление
- Вывод имеющихся товаров со стоимостью
- Покупка товаров
- Вывод информации:
    - Выручка
    - Проданные товары
    - Средний чек
